# q-workplatform

Frontend workspace for the Q PaaS platform, bootstrapped with [Ant Design Pro](https://pro.ant.design).

## Prerequisites

- Node.js 20+
- npm 10+

## Install

```bash
npm install
```

## Development

```bash
npm start
```

## Common Commands

```bash
npm run build
npm run lint
npm test
```

## Notes

This repository is intended to be managed as a submodule inside the `q-paas-studio` workspace.
